# Conversor de Moedas

Aplicação desenvolvida para praticar HTML, CSS e JavaScript através da criação de um conversor de moedas com interface intuitiva e atualização dinâmica dos valores convertidos.

## 📸 Preview

<img src="./assets/preview.png" alt="project-preview">

## 🚀 Tecnologias

* HTML5
* CSS3
* JavaScript

## 📚 Aprendizados

Durante o desenvolvimento deste projeto, pratiquei:

* Manipulação do DOM
* Eventos em JavaScript
* Estruturas condicionais
* Formatação de valores monetários
* Integração entre HTML, CSS e JavaScript
* Atualização dinâmica de elementos da página

## 🎯 Objetivo

Desenvolver uma aplicação funcional para consolidar conceitos fundamentais de JavaScript e fortalecer a construção de interfaces interativas.
